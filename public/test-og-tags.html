Тестовая страница для проверки OG tags

Эта страница создана для проверки метатегов Open Graph в Telegram и других социальных сетях.

Как проверить:

  1. Разместите эту страницу на вашем сервере
  2. Отправьте ссылку в Telegram
  3. Telegram должен показать большое изображение (1200x630px) с синим фоном и логотипом по центру

Также можно проверить через инструменты:

  • OpenGraph.xyz
  • Facebook Debugger
  • Twitter Card Validator
